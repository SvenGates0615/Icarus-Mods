## SG-Longer-Days_P.pak

Mod Version: 1

Compatible with Icarus Version: (Week: 146)

## Description:

Changes the length of day.

### Items Changed:

- Shorter nights.
- Longer days.
- Sunrise 6am.
- Sunset 8pm.
